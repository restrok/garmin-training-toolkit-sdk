# Garmin Connect Analyzer

## Use the main CLI instead:

```bash
python3 garmin.py collect      # Collect data
python3 garmin.py progress     # View stats
python3 garmin.py zones        # HR zones
python3 garmin.py recovery     # HRV/readiness
python3 garmin.py predict      # Race predictions
python3 garmin.py plan         # Generate plan
```
